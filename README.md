# Bath Boost
#### 技育CAMPハッカソン2024 vol.11最優秀賞受賞
友達と互いにお風呂報告をすることで、お風呂に入るモチベーションを高めるWebアプリケーションです。
チャットルームで、かわいいスタンプを使って目標宣言、お風呂報告ができます。
お風呂報告をするとお風呂ポイントが貯まり、お風呂レベルが上がります。
